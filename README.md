# keung-keung
졸작 / 디저트 탐색 지도(킁킁)

## 기술 스택

- **React + Vite**: 빠른 개발 환경과 HMR 지원
- **ESLint**: 코드 품질 관리

### 플러그인

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) - Babel을 이용한 Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) - SWC를 이용한 Fast Refresh
